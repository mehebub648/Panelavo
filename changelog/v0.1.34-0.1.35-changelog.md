# 0.1.34 to 0.1.35

- Bind the production Next.js process to `127.0.0.1:10443` so only CloudPanel/Nginx can reach it.
- Harden cookies, origin validation, client-address handling, CSP, cross-origin headers, and HTTPS HSTS responses.
- Remove trust in browser-supplied `X-Forwarded-Host` and forwarding chains.

Production must expose Panelavo through its HTTPS CloudPanel vhost. Direct public access to port 10443 is no longer supported; use an SSH tunnel for emergency local access.
