# 0.1.1 to 0.1.2

- Publish the complete Node.js distribution under `/usr/local/lib/panelavo-node`.
- Point shared Node, npm, npx, and Corepack commands at the accessible runtime.
- Verify Node and npx as a non-root user before continuing deployment.

This fixes application builds failing with `npx: command not found` when root's nvm directory cannot be traversed by the CloudPanel site user.
