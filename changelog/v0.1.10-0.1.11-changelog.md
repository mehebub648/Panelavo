# 0.1.10 to 0.1.11

- Configured the panel Nginx vhost to accept file-manager upload requests up to 96 MiB.
- Limited individual uploaded files to 64 MiB in both the browser and CloudPanel bridge.
- Added a clear error when an upload exceeds the supported size.

After upgrading an existing installation, run `sudo bash setup.sh` once to apply and validate the Nginx upload limit.
