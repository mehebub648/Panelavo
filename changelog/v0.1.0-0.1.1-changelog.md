# 0.1.0 to 0.1.1

- Protect the active SSH client from the `sshd` fail2ban jail while setup runs.
- Preserve the live session's actual SSH port before enabling or changing UFW.
- Document evidence-based SSH timeout diagnosis and console recovery.

The fail2ban exception is temporary by default. Fixed trusted administrator addresses can opt into retaining it with `KEEP_SSH_CLIENT_ALLOWLISTED=true`.
