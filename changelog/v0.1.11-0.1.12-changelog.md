# 0.1.11 to 0.1.12

- Added a Super Admin update manager to Settings.
- Added update checks and staged installation from a configurable public HTTPS Git repository.
- Preserved `.data` and `.env.local` during updates and reload only Panelavo, leaving hosted websites running.
- Added persistent update locking, status, logs, and safe failure reporting.

The default source is `https://github.com/mehebub648/Panelavo.git` on `main`. The updater runs without root privileges and does not apply host-level migrations.
