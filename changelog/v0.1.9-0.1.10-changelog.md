# 0.1.9 to 0.1.10

- Made SSH Git connections non-interactive and safely record new host keys on first use.
- Allowed cloning into a site root that contains only ACME's `.well-known` directory, preserving its challenge files.
- Allowed repository operations up to five minutes instead of the bridge's short default timeout.
- Replaced the generic 502 response with useful errors for non-empty roots, authentication failures, and missing repositories or branches.

No configuration migration is required. Private repositories still need the website's public deployment key.
