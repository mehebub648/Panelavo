# 0.1.2 to 0.1.3

- Pause only fail2ban's `sshd` jail during SSH-based provisioning.
- Restore the jail automatically on successful, failed, or interrupted setup exits.
- Retain an opt-in single-IP guard for stable administrator addresses.
- Document a provider-console recovery sequence for already-banned clients.
- Restore an sshd jail that was explicitly pre-paused from the provider console.

This makes installation reliable for mobile and carrier-NAT connections whose public address can change during maintenance.
